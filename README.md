# WaveTracker

Phase0 bootstrap for the Wi-Fi survey platform.

## Scope (Phase0)

- Development harness for backend, frontend, and infra
- Collector scripts for Windows and macOS
- Workspace-shared Copilot skill and instructions
- Setup and development docs

## Quick Start

1. Read `docs/SETUP.md`
2. Start local stack with infra compose
3. Run backend/frontend lint and tests
